Reliability Analysis of Io VT Based Intelligent Video Surveillance System
This paper proposes a different framework to analyze the system via the concept of reliability. The existing approaches depend solely upon the system input sequences. However, the proposed framework works on the component statistical index. Based on exponential distribution theory of reliability, the reliability of the intelligent VSS is analyzed in this paper, followed by a complete guideline to improve the current reliability of the system
Tanin Sultana, Mohammad Wajih Alam, Khan Wahid
